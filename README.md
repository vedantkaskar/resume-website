# Vedant Portfolio (Next.js 15)

A cinematic, premium personal portfolio for **Vedant Prashant Kaskar** with cloud-native and AI-first design language.

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- GSAP + ScrollTrigger
- React Three Fiber / Drei
- ShadCN-style reusable UI patterns
- Lucide icons

## Features

- Dark theme by default + theme toggle
- Animated gradients + glassmorphism UI
- 3D particle hero background
- Rotating roles + cinematic hero
- Interactive about cards + counters
- Orbit-style tech stack visualization
- Filterable premium project showcase
- Scroll-triggered timeline animation
- Cloud/DevOps dashboard section
- Command palette (`Cmd/Ctrl + K`)
- Keyboard navigation (`j` / `k`) + easter egg (`Shift + ~`)
- AI assistant UI mock
- Smooth inertial scrolling + cursor glow + scroll progress bar
- Responsive desktop/mobile layout

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Deploy On Vercel

1. Push this repository to GitHub.
2. Import the repo in Vercel.
3. Keep default framework settings (`Next.js`).
4. Deploy.
